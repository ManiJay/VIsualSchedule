# VIsualSchedule

This is a schedule Application that focuses on Austistic users and their guardian in managing their daily activities. 
The application comes with different interface like the profile, activities, settings and the help page. 
## Profile 
this helps in personalising and recognition of a user profile 
### Activities 
A user activities can be personalised with uploaded activities of their daily choice
## Settings and help page
helps our users with navigation of the webpage, creating a user friendly application.
